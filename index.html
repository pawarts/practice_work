Fullstack development

Full-Stack developer – це розробник, який бере участь у всіх етапах розробки веб-додатків. Він розуміє все, що потрібно для створення програми, і Frontend, і Backend.

Frontend
FrontEnd розробник створює видиму для користувача частину веб-сторінки і його головне завдання – точно передати у верстці те, що створив дизайнер, а також реалізувати логіку користувача.
Backend
Back-end (зворотний бік) developer) — це фахівець, який займається програмно-адміністративною частиною веб-додатків внутрішнім змістом системи, серверними технологіями — базою даних, архітектурою, програмної логікою.

Frontend development

Важливу роль у житті кожного Fullstack розробника відіграє frontend розробка. Більшість сучасних розробників починають свою кар’єру саме з неї. Саме завдяки frontend розробникам сайти стають усе красивіші та функціональніші.

Які мови потрібно знати Front end розробнику?

HTML5
HTML (HyperText Markup Language) -стандартизована мова розмітки документів для перегляду веб-сторінок у браузері.
CSS3
Каскадні таблиці стилів (англ. Cascading Style Sheets - CSS). Мова для запису стильового оформлення веб-сторінок. Колір тла, розмір літер та найрізноманітніші візуальні ефекти створюються за допомогою CSS.
JavaScript
JavaScript було створено для того, щоб “оживити веб-сторінки”. Програми на цій мові називаються скриптами. Їх можна писати прямо на сторінці в коді HTML і вони автоматично виконуються при завантаженні сторінки.
